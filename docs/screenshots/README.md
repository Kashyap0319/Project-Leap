# Screenshots

This directory contains screenshots of the Project-Leap dashboard.

## Images to Add

1. **dashboard-overview.png** - Main dashboard showing KPIs, charts, and metrics
2. **services-page.png** - Services page with analytics and endpoint details
3. **logs-page.png** - Live traffic logs with filtering options
4. **dashboard-metrics.png** - Complete dashboard view with all widgets

## How to Add Screenshots

1. Save your screenshots with the names above
2. Place them in this `docs/screenshots/` directory
3. The README.md will automatically display them

## Screenshot Guidelines

- **Format:** PNG or JPG
- **Size:** Recommended 1920x1080 or similar
- **Naming:** Use exact names as listed above
- **Content:** Should show the actual working dashboard
